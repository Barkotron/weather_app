# Weather App

Uses [Open-Meteo](https://open-meteo.com/) API for weather data

Gives hourly forecast data for a 7-day period

### Dependencies
```
"framer-motion": "^11.2.6",
"prop-types": "^15.8.1",
"react": "^18.2.0",
"react-dom": "^18.2.0",
"react-lottie": "^1.2.4"
```

### Usage

```
npm install
npm run dev
```
